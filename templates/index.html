MIRA3

Bienvenido a la aplicación web creada para la asignatura de Ingeniería de Sistemas de Información del grado en Ingeniería Informática de la UGR. Esta aplicación esta construida con Python, Flask, JavaScript, HTML y Boostrap. Nos centramos en herramientas y técnicas de obtención de datos como las APIs y el Web-Scraping. Esta aplicación en concreto es para buscar un título de una pelicula y obtener el rating en varios sitios webs y el trailer de Youtube. Espero que le sea útil.
